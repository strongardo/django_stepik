My dictionary

My Dictionary is a simple Django program that provides a web interface for creating and managing a personal dictionary. The website includes three pages:

  1. Home Page – a brief description of the program.
  2. Words List – displays the dictionary words in a table, with the first column showing English words and the second column their Russian translations.
  3. Add a Word – a page with a form to input a new word (in English) and its translation (in Russian), along with a "Submit" button to save it.

The program is perfect for learning and practicing vocabulary.
